## Sections

### Home
The home section provides a brief introduction about myself, including links to my social media profiles. It also features two prominent buttons: 
- **Hire Me**: This button directs potential clients to contact me for hiring inquiries.
- **Download CV**: This button allows visitors to download my curriculum vitae.

### About
The about section offers more detailed information about me, including my educational background and work history. 

### Services
In the services section, I outline the various services I can provide to clients, showcasing my skills and expertise.

### Projects
The projects section displays my most recent project works, providing visitors with insight into my portfolio and capabilities.

### Contact
The contact section enables visitors to reach out to me directly. It includes fields for the subject and message, along with a "Let's talk" button. Clicking this button will open the default email client with my email address pre-filled in the "to" field, and the subject and message pre-written based on the input provided on the website.

## Installation
As this is a static website built with Flutter, you can simply clone this repository and run it using any Flutter-compatible development environment.

## Usage
Feel free to use this code as a template for your own portfolio website. Customize it with your own content, projects, and services to showcase your skills and attract potential clients.
